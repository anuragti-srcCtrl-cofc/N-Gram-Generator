# N-Gram Text Predictor

A web-based interactive tool that generates text based on an N-Gram model from a user-provided corpus.

## Features
- Paste any text corpus.
- Select N-gram size (3 to 15).
- Select output text length.
- Generates text using the built N-gram model.
- Beautiful, modern UI.

## Usage
Simply open `index.html` in your browser or visit the GitHub Pages link once deployed.
